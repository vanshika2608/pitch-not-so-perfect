# Pitch-Not-So-Perfect™ 🚀

Generate the next billion-dollar startup nobody asked for.

An AI-powered startup satire generator that creates absurd Silicon Valley startup pitches filled with buzzwords, fake valuations, visionary founders, and investor delusion.

## Features

- AI-generated fake startups
- Startup names + slogans
- Ridiculous investor pitches
- Fake funding rounds & valuations
- Founder lore generation
- Luxury dark-mode UI
- Real-time AI responses using Groq

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Groq API
- Llama 3.1

## Example Output

```txt
NAME:
EchoPlex

TAGLINE:
"Experience a harmonious blend of human and AI-driven wellness."

VALUATION:
$42M Series B

FOUNDER:
AI Ethnologist & Meditation Thought Leader
```
